Development and Investigation of a
Mobile Forensics Testbed for Educational Use

This project was carried out in my 4th year at Abertay University for my honours project in Ethical Hacking.

The aim of this project was to analyse current issues, tools and methods encountered in mobile forensics. This analysis would then be used to compare the amount and variety of data that can be obtained from two different developed test beds with the variance of one device being rooted and the other non-rooted
